# Cloud Inventory Manager

# Project Description

This is a simple inventory management system built using Python Flask and SQLite.

The project is used to store and manage product details such as product name, category, price, and quantity.

This project is based on the RDS concept, where data is stored and managed in a relational database.

# Technologies Used

- Python
- Flask
- SQLite
- VS Code

# Features

- Add Product
- View Products
- Update Product
- Delete Product
- Search Product
- Low Stock Alert
- Product Count

# Database Fields

- id
- product_name
- category
- price
- quantity
